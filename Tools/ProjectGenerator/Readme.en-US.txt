== Tool: Project generator
This is a command line tool to generate and update project files.
Currently only Code::Block project file generation is supported.

= Prerequisites
The source depends on YSLib b599.

= Source configuration
There is currently no configuration supported.

= Usage
Call without options to see details.
Currently the output stream would always be raw UTF-8 (without any byte order mark).
It is recommended to backup or ensure the files are in version control system before any operations.
